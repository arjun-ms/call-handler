# Changelog - 2026-05-15T22:34

## Focus: Evaluation Tooling & Edge Case Discovery

### Changes
- Increased speech capture limit from 7s to 15s for better accuracy.
- Moved Speaker Diarization to Future Roadmap (not needed for MVP).
- Created standalone `scripts/run_evaluation.py` batch evaluation tool.
- Added multi-format support (.aac, .ogg, .webm, .opus, .m4a) to eval script.
- Removed `scikit-learn` from requirements.

### Edge Case Discovered
- Gender misclassification on teenage speakers: real 17-year-old female recording predicted as male. Age bracket (18-30) was correct. Root cause: acoustic overlap in young voices. Low risk for production (callers are adults).

### Test Results
- 14/14 core tests passing.
- CREMA-D: Gender 100%, Age 66.7% (3 samples).
